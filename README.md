# StopWatch Web Application

Prodigy InfoTech Internship Task 2: Stopwatch Web Application
This repository contains my submission for Task 2 of the Prodigy InfoTech Web Development Internship. The assignment was to develop a fully functional and visually appealing Stopwatch Web Application using core front-end technologies.

Project Overview

Task: Build a stopwatch web application as part of the Prodigy InfoTech internship program.

Objective: Design and implement a stopwatch that allows users to start, stop, reset, and (optionally) record lap times, with a clean and responsive interface.

Technologies Used: HTML, CSS, JavaScript.

Features

Responsive design for seamless use on desktop, tablet, and mobile devices

(Optional) Lap time recording and display for advanced functionality

Modern and user-friendly UI. ( file:- task2.html, bg1.jpg)

# Weather Application
Prodigy InfoTech Internship Task 5: Weather Application
This repository contains my submission for Task 5 of the Prodigy InfoTech Web Development Internship. The assignment was to create a fully functional and responsive Weather Application using modern web technologies and real-time weather data integration.

Project Overview

Task: Build a Weather Application as part of the Prodigy InfoTech internship program.

Objective: Design and implement a web app that fetches and displays real-time weather information for any city or the user's current location, with an intuitive and visually appealing interface.

Technologies Used: HTML, CSS, JavaScript, OpenWeatherMap API.

Features

Real-time weather updates by city search or current location

Display of temperature, humidity, wind speed, and weather conditions

Dynamic background or visuals based on weather/location

Search bar with button and enter-key functionality

Responsive and user-friendly interface for all devices. ( file :- index.html, script.js, test-api.js, package.json, package-lock.json)
